# Test Module Requirements

This file contains requirements to test whitespace preservation.

## Requirements

**SRS_WHITESPACE_TEST_03_001: [** test_module_create shall allocate memory for a test module. **]**

**SRS_WHITESPACE_TEST_03_002: [** test_module_create shall return NULL if allocation fails. **]**

**SRS_WHITESPACE_TEST_03_003: [** test_module_destroy shall free all allocated memory. **]**
