# Test Module Requirements

This file contains requirements to test line-style comment handling.

## Requirements

**SRS_LINE_COMMENT_TEST_02_001: [** test_module_create shall allocate memory for a test module. **]**

**SRS_LINE_COMMENT_TEST_02_002: [** test_module_create shall return NULL if allocation fails. **]**

**SRS_LINE_COMMENT_TEST_02_003: [** test_module_destroy shall free all allocated memory. **]**
