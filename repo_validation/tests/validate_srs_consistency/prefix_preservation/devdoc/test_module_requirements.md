# Test Module Requirements

This file contains requirements to test prefix preservation.

## Requirements

**SRS_PREFIX_TEST_01_001: [** test_module_create shall allocate memory for a test module. **]**

**SRS_PREFIX_TEST_01_002: [** test_module_create shall return NULL if allocation fails. **]**

**SRS_PREFIX_TEST_01_003: [** test_module_destroy shall free all allocated memory. **]**

**SRS_PREFIX_TEST_01_004: [** test_module_process shall validate the input parameter. **]**
